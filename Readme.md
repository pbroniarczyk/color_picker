# Color autocomple

Tiny React app for changing background color of entire application. Colors are fetched from outside API and displayed in autocomplete form.

### Launching app

Install depedencies

```
yarn
```

Start app locally ( it will automticly open browser tab )

```
yarn start
```

----------

## App tests

```
yarn test
```

----------

* ES6
* React
