# Color autocomple

Tiny React app for changing background color of entire application. Colors are fetched from outside API and displayed in autocomplete form.

### Launching app

Install depedencies

```
yarn
```

Start app locally ( it will open browser tab at: localhost:8080 )

```
yarn start
```

----------

Run tests

## Running the tests

```
yarn test
```

----------

### Technology stack

* ES6
* React
* Reduc
* Jest
* Enzyme
